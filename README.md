# Vellium Plugins Repository

This repository hosts a complete, curated collection of plugins for high-quality media streaming and structured emulator support.

## 📦 Plugin Categories

### 🎬 Media Plugins
- **Movies**: Meta Movies, 4K HDR, Dolby Vision/Atmos
- **TV**: Meta TV, Season Packs
- **Anime**: Nyaa-based with Sub/Dub/Raw support
- **HQ**: Seeder-prioritized, Cached-Optimized
- **Ultimate**: All-in-one strongest sources

### 🎮 ROM & BIOS Plugins
- **Starter Kits**: RetroArch, Full Emulation
- **BIOS**: All Systems, Sony, Sega, Arcade, Handheld
- **Consoles**: Nintendo, PlayStation, Sega, Xbox
- **Arcade**: MAME, FBNeo
- **Formats**: CHD Only, Cartridge Only
- **Meta**: ROMs HQ

## 🚀 Quick Start

### Using with Vellium App

1. Open Vellium app
2. Go to Settings → Plugins
3. Tap "Sync Plugins"
4. Enable the plugins you want to use

### Manual Installation

1. Download the desired YAML files
2. In Vellium, go to Settings → Plugins → Add Plugin
3. Import the YAML file
